SSH-Befehle zum Ausführen auf dem Server:

1. Verbinde dich per SSH:
   ssh adm_ssh@192.68.4.59

2. Wechsle zum Service-Verzeichnis:
   cd /volume1/nodejs/UricAcidService

3. Installiere die Dependencies:
   npm install express-session passport passport-google-oauth20

   ODER alle Dependencies:
   npm install

4. Prüfe ob die Installation erfolgreich war:
   ls node_modules | grep -E "(express-session|passport)"

5. Starte den Service neu (falls er läuft):
   pm2 restart UricAcidService
   # oder
   node src/app.js
